Public-Data Consoles

Self-contained, browser-only dashboards built on live public data feeds. Each is a single HTML file — open it and it runs; no install, no account, no server.

These are descriptive, decision-support tools, not predictions. Data sources are public and labeled on each console (measured / modeled / demo). Nothing here contains protected health information or proprietary data. Readings are coincident indicators — a high value means "large/coupled today," not a forecast.
