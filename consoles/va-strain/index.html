Operator hook — wire your live feeds

How to make this real (no code edits to panels)

Real-time strain (census, ED boarding, RN staffing) is internal to VHA and not public, so this console ships with clearly-labeled DEMO data. To make it live, push your facility's numbers into the global override object from your own scheduled job / dashboard backend — no edits to the panels needed:

STATUS_OVERRIDE['ICU'] = {census:18, cap:16, rnAct:7, rnTgt:8, tier:'alert'};
STATUS_OVERRIDE['ED'] = {board:22, lwbs:6.0, divert:true}; then call render().

Keys are the unit IDs shown in the ranking (ED, ICU, 3W, 4E, MH, CLC, OR). Any field you supply overrides the demo value; fields you omit keep the demo. A tier:'alert'|'watch' flag forces that unit's escalation. Public anchors (VA Access-to-Care wait times, facility list/quality via api.va.gov Facilities API — note: that API needs a free key) are lagged and best baked as the baseline, with the operator hook carrying the live tail.
