About & honesty. This reorganizes public data into one response picture for any significant quake worldwide. Badges mark what each figure is: measured (sensors/observations), modeled (estimates with uncertainty — e.g. PAGER fatalities/loss), interpretive (inferred, e.g. the lifeline cascade on approximate dependency data). It never invents casualty counts. It does not replace official agencies or on-the-ground assessment — it aims response sooner.
